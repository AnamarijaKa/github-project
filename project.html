GitHub Vodič

Kreiranje i upravljanje granama (branches)

Grane vam omogućuju paralelni razvoj bez utjecaja na glavnu granu.

git checkout -b naziv-grane

Kreiranje i pregled Pull Request-ova (PR)

  1. Pushajte promjene: git push origin naziv-grane
  2. Kreirajte PR putem GitHub sučelja
  3. Dodajte opis, recenzente i kliknite "Create pull request"

Korištenje Issues za praćenje zadataka i komentara

Issues služe za bugove, prijedloge i rasprave.

  • Kreirajte novi issue
  • Dodajte naslov, opis, label i assignee
  • PR može zatvoriti issue automatski: Closes #1
